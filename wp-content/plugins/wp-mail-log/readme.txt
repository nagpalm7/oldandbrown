WP Mail Log helps you to Log and view all emails from WordPress. It is useful if you have to debug email related problems or have to store sent emails for auditing. It provides an easy and simple user interface in the WordPress Admin.

== Description ==

WP Mail Log is a WordPress plugin that allows you to easily log and view all emails that were sent from WordPress. This would be very useful for debugging email related problems in your WordPress site or for storing sent emails for auditing purposes, especially on eCommerce sites.
